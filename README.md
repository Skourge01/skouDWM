# skouDWM
minha configuracao do dwm para a melhor rapidez

### Dwm - window manager dinamico 
- - - 
#### um bom window manager rapido e pequeno para X 

- - - 
- esta e a minha config de dwm com minhs opcoes de configuracao dando forked do [dwm da suckless](https://dwm.suckless.org/)

## Requisitos:
- - - 
- atualize seu sistema para instalar ( em arch linux )
sudo pacman -Syu 
## e instale as dependencias: 
sudo pacman -S xorg-server xorg-xrandr xorg-xinit libx11 libxft libxinerama 

- - - 
## Instalação 
- - - 
- clone o repositorio e compile:
git clone https://github.com/Skourge01/skouDWM.git

apos isso coloque o dwm no xinitrc
